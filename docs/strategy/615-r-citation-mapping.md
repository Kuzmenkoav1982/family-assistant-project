## Использование документа

**Для презентации инвестору:** показывайте полную таблицу Раздела V — это и есть «рынок с государственной поддержкой».

**Для презентации региону:** выберите 3–5 строк, релевантных конкретному региону (особенно если регион получает софинансирование за низкий СКР).

**Для презентации работодателю:** строки из задачи 5 + соответствующие KPI. Плюс — упоминание Всероссийского конкурса «Российская организация высокой социальной эффективности».

**Для пилота с женскими консультациями:** строки из задачи 3 + KPI «беременные в репрод. выборе → встали на учёт».
