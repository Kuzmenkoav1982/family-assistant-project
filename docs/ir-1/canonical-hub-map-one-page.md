# Canonical Hub Map — One Page
## IR-1 v0.1 | Baseline before investor meeting

---

## Platform model

> Платформа — семейная ОС.  
> Хабы — контуры жизни семьи, не просто разделы меню.  
> Домовой — сквозной семейный помощник поверх всех хабов.  
> RAG — контекстный движок: память, знание о семье, подложка для ИИ-слоя.

---

## Hub map

### Home Hub
**Роль:** главный семейный cockpit, обзор и вход в день

| | |
|---|---|
| Входит | дашборд семьи, сводка событий дня, быстрый доступ к активным задачам, уведомления, семейный пульс |
| Не входит | управление структурой семьи, глубокие профили, архивные данные |
| Ключевые сценарии | утренний обзор дня / недели; быстрый переход в нужный хаб; видеть, что важно прямо сейчас |

---

### Family Hub
**Роль:** семья как структура — люди, роли, связи, доступы, дерево

| | |
|---|---|
| Входит | профили членов семьи, семейное дерево, роли и доступы, связи между членами |
| Не входит | планирование событий, developmental record детей, финансы |
| Ключевые сценарии | добавить/обновить профиль члена семьи; посмотреть семейное дерево; управлять ролями и доступами |

---

### Planning Hub
**Роль:** оперативное исполнение — задачи, календарь, покупки, планы

| | |
|---|---|
| Входит | календарь событий, задачи, покупки (execution), план покупок (намерения), общие планы |
| Не входит | жизненные цели, финансовое планирование, развитие детей |
| Ключевые сценарии | запланировать событие / задачу; вести список покупок; видеть неделю семьи |

**Граница внутри хаба:**
- Календарь = оперативные события
- Праздники = культурный / семейный повторяющийся слой (может жить в Values Hub или как подтип в Planning)
- Покупки = execution, уже решённые
- План покупок = intention layer, будущие приобретения

---

### Development Hub
**Роль:** рост человека и семьи — портфолио, мастерская, векторы

| | |
|---|---|
| Входит | Портфолио развития, Мастерская жизни, Развитие (планы/действия/привычки), детский developmental record |
| Не входит | оперативные задачи, финансовые цели, ежедневный планировщик |
| Ключевые сценарии | зафиксировать достижение / веху; посмотреть траекторию развития ребёнка; поставить долгосрочный вектор роста |

**Граница внутри хаба:**
- Портфолио развития = диагностика / зеркало / интерпретация прогресса
- Развитие = планы / действия / привычки / исполнение
- Мастерская жизни = смысловой и биографический контур, долгие векторы, семейная летопись

---

### Health Hub
**Роль:** здоровье семьи — данные, наблюдения, события

| | |
|---|---|
| Входит | здоровье членов семьи, медицинские события, наблюдения, метрики здоровья |
| Не входит | финансы на лечение, общий календарь |
| Ключевые сценарии | записать визит к врачу; отследить самочувствие; хранить медицинскую историю |

---

### Household Hub
**Роль:** быт, дом, имущество, хозяйственные процессы

| | |
|---|---|
| Входит | имущество, дом, бытовые задачи, хозяйственные процессы, документы на имущество |
| Не входит | общий планировщик, финансы, семейная структура |
| Ключевые сценарии | вести реестр имущества; фиксировать бытовые задачи; хранить документы на недвижимость |

---

### Leisure Hub
**Роль:** отдых, поездки, досуг, впечатления

| | |
|---|---|
| Входит | путешествия, планирование отдыха, досуговые события, wishlist поездок |
| Не входит | ежедневный планировщик, финансы, семейная структура |
| Ключевые сценарии | запланировать поездку; сохранить идею для отдыха; вести историю путешествий |

---

### Values Hub
**Роль:** ценности, традиции, культура, вера, смыслы семьи

| | |
|---|---|
| Входит | традиции, праздники (культурный слой), семейные ценности, ритуалы, вера |
| Не входит | оперативный календарь, задачи, финансы |
| Ключевые сценарии | зафиксировать семейную традицию; отметить важную дату / праздник; описать ценности семьи |

---

### State Hub
**Роль:** состояние семьи — метрики, сигналы, статусы

| | |
|---|---|
| Входит | финансовое состояние, жизненные цели, финансовые цели, семейные метрики |
| Не входит | оперативные задачи, developmental record |
| Ключевые сценарии | поставить жизненную цель семьи; отслеживать финансовое состояние; видеть общий "индикатор здоровья" семьи |

**Граница внутри хаба:**
- Цели = верхнеуровневые жизненные цели семьи / человека
- Финансовые цели = подмножество с деньгами, сроками, накоплением

## Platform layers (not hubs)

### Domovoy — Family Copilot
> Сквозной семейный помощник поверх всех хабов.

- Не отдельный раздел, а продуктовый слой
- Напоминает, собирает повестки, подсказывает важное
- Помогает превращать события в ритуалы, планы и память
- Соединяет разные хабы в единый поток

**Статус:** Vision / roadmap. Не обещаем в ближайшем релизе как готовый продукт.

---

### RAG — Family Context Engine
> Инфраструктурный интеллект-слой: память, контекст, знание о семье.

- Не пользовательский раздел — невидимый движок
- Поиск и извлечение семейного контекста из событий, документов, истории, целей
- Подложка для Домового, AI-инсайтов, умного поиска, напоминаний
- Позволяет ИИ отвечать не "вообще", а в контексте этой конкретной семьи

**Статус:** Vision / roadmap. Позиционируется как следующий слой интеллекта платформы.

## Investor framing in 3 sentences

Мы строим семейную ОС — не набор разрозненных экранов, а систему контуров жизни семьи.  
Сегодня работают ключевые прикладные хабы: структура, планирование, развитие, быт, здоровье, ценности.  
Следующий шаг — превратить платформу из набора разделов в контекстно-умную систему через RAG и Домового.
