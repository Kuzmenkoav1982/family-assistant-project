## Draft canonical direction

### Development cluster
| Сущность | Роль |
|----------|------|
| Портфолио развития | диагностика / зеркало / интерпретация прогресса |
| Развитие | планы / действия / привычки / исполнение |
| Мастерская жизни | смысловой и биографический контур, долгие векторы, семейная летопись |

### Goals cluster
| Сущность | Роль |
|----------|------|
| Цели | верхнеуровневые жизненные цели семьи / человека |
| Финансовые цели | подмножество целей с деньгами, сроками, накоплением |

### Planning cluster
| Сущность | Роль |
|----------|------|
| Календарь | оперативное планирование и события |
| Праздники | культурные / семейные / повторяющиеся значимые даты |
| Покупки | фактические покупки / execution |
| План покупок | подготовленный список / намерения / будущие приобретения |

### Children cluster
| Сущность | Роль |
|----------|------|
| Дети | отдельный семейный контур ребёнка: профиль, режим, события, потребности |
| Портфолио развития | developmental record ребёнка как часть траектории |
| Развитие | действия, планы, занятия, треки роста |
